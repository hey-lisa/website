$LISA est le utility token qui permet l'accès à LiSA et son écosystème de fonctionnalités.

### Concept principal
- **Hold-to-access model** : Vous devez détenir des tokens $LISA pour utiliser LiSA
- **Utility token** : Pour l'accès, la création et participation aux strategies, et les mécanismes de fees
- **Standard ERC-20** : Construit sur Ethereum pour une large compatibilité


Pour des informations détaillées sur les mécanismes du token, les exigences, et les cas d'usage spécifiques, consultez la [section dédiée au token $LISA](/docs/lisa-token/summary).
