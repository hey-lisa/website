### Comment lier dans le chat
- Demandez à LiSA.
- LiSA affiche un one time link code, à copier
- Sur l'autre wallet, connectez-vous et demandez à LiSA
- Signez une one time proof dans chaque wallet pour confirmer le contrôle.
- Une fois liées, les deux adresses apparaissent sous le même profil.

### Gérer
- Demandez à LiSA : liste les linked addresses / adresses liées
- Demandez à LiSA : delie/unlink 0xABCD… pour en retirer une
- Ajoutez des labels et de courtes notes pour le contexte.
- Ou dans les options

### Sécurité
- Le linking ne donne aucune permission de dépense.
- Chaque action requiert toujours une signature depuis l'adresse qui détient les fonds.
