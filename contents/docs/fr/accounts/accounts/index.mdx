### Créer un compte
- Pour créer un compte, il suffit de connecter votre wallet
- Aucune inscription préalable requise
- L'authentification se fait par signature de message
- Votre adresse devient automatiquement votre identifiant 

### Profils : lier plusieurs adresses sous le même compte

Vous pouvez regrouper plusieurs de vos wallets sous un seul profil LiSA (voir section suivante)


**Avantages** :
- Envoi de tokens entre vos wallets en mode restricted
- Stratégies et portfolios regroupés sous un seul profil
- Gestion centralisée de tous vos assets

### Paramètres du profil

**Configuration disponible** :
- API key pour Hyperliquid
- Options d'UI/UX
- Adresse email (optionnelle)
- etc.

**Notifications email** :
- Laissez LiSA vous envoyer des notifications
- Rapports de performance des stratégies  
- Alertes sur vos positions

### Connexion et sécurité

- Connectez-vous en signant un message d'authentification one time
- C'est off chain et cela coûte zéro gas
- Vous signez chaque action dans votre wallet
- LiSA ne demande jamais de seed phrase ni de private key
