### Identification dans le chat
- Les symboles et les noms sont reconnus dans le chat.
- Vous pouvez coller une adresse de contrat pour spécifier un token custom ou non listé.

### Approvals
- Le premier usage d'un token peut nécessiter une approval vers un spender (comme 1inch LiFi etc.).
- Les approvals sont par token et par blockchain et peuvent être revoke à tout moment.
