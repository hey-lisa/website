### Lock pour créer une stratégie

Pour publier une stratégie et la maintenir active :
- Vous devez locker un montant minimum de $LISA 
- Le lock garantit votre engagement en tant que créateur
- Votre stratégie reste visible tant que le lock est maintenu

### Lock pour participer

**En tant que participant** :
- Vous lockez du $LISA pour rejoindre des stratégies partagées
- Le montant à locker est déterminé par le créateur de la stratégie  
- Vous pouvez unlock selon les conditions définies par le créateur

**En tant que créateur** :
- Vous lockez du $LISA pour publier vos stratégies
- Vous gardez les profils de stratégie actifs grâce au lock
- Vous percevez les splits sur les $LISA lockés par les participants

### Mécanisme de lock

- **Durée** : Définie lors de la création de la stratégie
- **Early unlock** : Possible avec pénalités selon les règles
- **Flexibilité** : Conditions personnalisables par stratégie
