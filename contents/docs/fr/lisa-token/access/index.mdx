### Exigence de hold
- Un solde minimum de $LiSA est requis pour accéder à tous les services
- Montants précis à définir selon les services

### Hold effectif
- Votre hold effectif = $LISA dans votre wallet + $LISA lockés sous votre adresse
- Tous vos locks (stratégies créées + participations) comptent dans le calcul
