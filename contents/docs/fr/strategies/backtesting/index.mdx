### Backtesting de stratégies
- Chaque stratégie, une fois compilée, est exécutée sur des données historiques.
- Vous accédez aux résultats historiques RÉELS de vos stratégies
- Simule sur des données historiques avec les mêmes outils utilisés en live.
- Affiche des données techniques : PnL, drawdown, Sharpe ratio, maximum drawdown duration, win rate, profit factor, etc.
