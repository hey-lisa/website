### Hyperliquid
- Utilise votre compte via API.
- LiSA place les ordres. Les clés restent chez vous.

### On chain
- Utilise nos vaults on chain.
- Vos fonds sont dans un contrat sur lequel LiSA peut agir de façon limitée et contrôlée.
- LiSA (qui possède un wallet) place des swap, mais ne peut pas withdraw()
- Le vault est codé de sorte que même si le wallet de LiSA était sous contrôle d'un acteur malveillant, vos fonds resteraient en sécurité.
- Le seul moyen pour qu'un acteur malveillant accède à vos fonds, est que votre propre wallet soit corrompu.
