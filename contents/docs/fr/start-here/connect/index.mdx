title: Se Connecter
description: Connectez votre wallet à LiSA via wallets compatibles EIP-1193 / EIP-6963. Authentification one-time sans gas fees.
---

### Étapes
- Cliquez sur Connect, choisissez votre wallet.
- Signez le message d'authentification one time dans votre wallet pour démarrer la session. 
- L'auth est off-chain et ne coûte aucun gas.

### Ce que fait la signature
- Prouve que vous contrôlez l'adresse pour cette session.
- Crée une session qui se termine quand vous vous déconnectez ou effacez les données du site.
- Cela ne donne pas de token approvals et ne déplace pas de fonds.
- LiSA ne peut absolument jamais toucher à vos fonds
