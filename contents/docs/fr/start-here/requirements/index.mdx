### Wallet
- Pour accéder à LiSA, il faut se connecter à la dApp.
- Les wallet compatibles EIP 1193 / EIP 6963.

### Réseaux et gas
- Les principales blockchains EVM sont supportées.
- Gardez une petite quantité de native gas sur la blockchain où vous agissez.

### Accès avec $LISA
- Accéder à LiSA exige de hold un solde minimum de $LISA.
- Le hold effectif est égal aux $LISA sur votre wallet + le total des $LISA lockés dans des stratégies 

### Connecteurs de trading
- Hyperliquid : API key.
- On-chain : les fonds sont déposés dans un smart contrat (vault)
