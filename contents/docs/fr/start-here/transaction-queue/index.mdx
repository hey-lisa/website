---
title: Transaction Queue (TX Queue)
description: Transaction queue pour relire et signer les transactions préparées. Transaction management.
---

La TX queue est une partie importante de l'UI : elle liste les transactions préparées et non signées créées à partir de votre demande. 
Vous les relisez et les signez une par une, rien ne s'exécute sans votre signature. 
Chaque étape se déverrouille seulement après la confirmation on chain de l'étape précédente.

Voir Transaction queue dans Moves pour plus de détails.
