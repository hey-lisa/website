title: First actions
description: Your first steps in LiSA
---

 
### Moves, from intent to queue
- Say what you want in chat, for example swap 1 ETH to USDC on Arbitrum.
- She returns a quote with route, fees, and minimum received, then prepares the transaction in the queue.
- Review the item and sign in your wallet. Nothing sends without a signature.
- If an approval is needed, you sign it once before the action.
- You can remove items from the queue before signing.

### Strategies, from idea to live
- Describe the plan in chat.
- She compiles it to a sealed spec. You can backtest before running.
- Choose where it runs, on-chain or on Hyperliquid.
- Start the run. She places orders. You monitor positions and fills. You can pause or edit.

### Optional setup
- Add a recipient whitelist if you want guardrails on sends.
- Link your Hyperliquid account by API if you plan to run strategies there.

### Safety
- LiSA never holds funds or moves tokens by herself.
- Every action is a prepared transaction that you sign.
