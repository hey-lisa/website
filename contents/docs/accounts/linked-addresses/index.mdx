---
title: Linked addresses
description: Link multiple addresses to one profile
---

 
### How to link in chat
- Ask LiSA: link another address.
- LiSA shows a one time link code.
- On the other wallet, connect and say: link with code ABCD.
- Sign a one time proof in each wallet to confirm control.
- When linked, both addresses appear under the same profile.

### Manage
- Ask LiSA: list linked addresses.
- Ask LiSA: unlink 0xABCD… to remove one.
- Add labels and short notes for context.

### Safety
- Linking does not grant spend permissions.
- Every action still requires a signature from the address that holds the funds.
